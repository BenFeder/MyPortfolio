  • Summarist

    Html, CSS, JavaScript, React, Firebase

    Summarist is a book reading web application that helps users read and retain more information from books by providing chapter summaries, key takeaways, and author and book details. Audio recordings of premium books for subscribed and authenticated users are available. I built the frontend for the entire application using React and Firebase for authentication and subscription management. Book details such as summary, description, etc. were retrieved from book APIs utilizing axios.

I maintained and debugged, adding numerous features to the site that polished its frontend and let the site shine.   • Movie Search App

The project involved integrating APIs, firebase for user authentication, and managing state effectively to provide a seamless user experience.
